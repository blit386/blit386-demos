# American English spelling

Canonical reference: [CLAUDE.md](../../CLAUDE.md) (Documentation Style).

Demo comments and identifiers use American English: `color`, `center`, `canceled`, `traveling`, `gray`. Never `colour`,
`centre`, `cancelled`, `travelling`, `grey`. Exempt: literal third-party or spec-mandated names correctly spelled with a
British `s`/`c` in their own spec (for example Web Audio's `AnalyserNode`/`createAnalyser`) – do not "fix" those if a
future demo references them.

Cursor: `.cursor/rules/american-english-spelling.mdc` (always applied in this repo).
